## To play around with
### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/risharma101/risharma101.github.io
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Launch on localhost
   ```sh
   npm start
   ```
### Deploy
   ```sh
   npm run build
   ```
